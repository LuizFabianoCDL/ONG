# ONG — Projeto (Entrega Final)

**Resumo:**  
Este repositório apresenta a versão final do projeto da disciplina, com práticas profissionais de versionamento (GitFlow), acessibilidade (WCAG 2.1 AA), otimizações para produção e documentação técnica exigida pela entrega.

---

## Status do repositório
- Branch principal (produção): `main`
- Branch de desenvolvimento: `develop`
- Estratégia: GitFlow (`feature/*`, `release/*`, `hotfix/*`)

---

## Tecnologias (exemplo)
- HTML5, CSS3, JavaScript (ES6+)
- Ferramentas de build: npm, html-minifier, terser, cssnano
- Deploy recomendado: Vercel, Netlify ou GitHub Pages

---

## Como rodar localmente (exemplo)
1. Instalar dependências:
```bash
npm install
```
2. Rodar em modo de desenvolvimento:
```bash
npm run dev
```
3. Gerar build de produção (minificado):
```bash
npm run build
```
